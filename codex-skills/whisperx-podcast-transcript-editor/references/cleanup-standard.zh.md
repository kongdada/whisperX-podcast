# 忠实清洗标准

把播客转写稿整理成更适合阅读的版本，但必须忠实于原文对话。

## 硬性规则

- 只做忠实清洗，不总结，不解释，不扩写，不删改观点。
- 保持说话人顺序、时间范围、段落顺序不变。
- 保留每个说话人标题行，格式不要改。
- 只修明显错字、明显专有名词错误、标点、断句和轻微语病。
- 不确定的人名、机构名、专有名词，不要擅自改写。
- 保留口语感，不要改成书面腔。
- 遇到明显噪声、乱码、重复串，可以删除，但要非常克制。
- 输出只包含清洗后的正文内容，不要额外写说明。

## 优先检查项

1. 明显 ASR 错字
2. 专有名词的高置信修正
3. 标点补齐
4. 单人长段的断句和分段
5. 重复噪声或明显无意义短语

## 保守处理规则

遇到下面这些情况，优先保守：

- 说话人可能贴反，但证据不足
- 人名或机构名有多个可能写法
- 一句话明显缺词，但无法从上下文唯一确定
- 某些术语像是识别错误，但无法高置信还原

这类情况宁可少改，也不要脑补。
